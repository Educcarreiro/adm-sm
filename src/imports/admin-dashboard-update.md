1️⃣ Tornar o Dashboard Mais Operacional

Adicionar uma seção logo abaixo dos indicadores chamada:

Ações Rápidas

Criar botões de ação rápida:

+ Cadastrar Novo Cliente
+ Adicionar Usuário Interno
Ver Chamados Abertos
Gerenciar Upsells

Esses botões devem abrir os respectivos modais ou páginas.

2️⃣ Corrigir Botão "Cadastrar Novo Cliente"

Atualmente o botão não está funcionando.

Ao clicar deve abrir um modal de cadastro de cliente.

Campos necessários:

Nome do Clube
Plano (Starter / Premium / Enterprise)
Valor Mensal (€)
Responsável do Clube
Email do Responsável
Número de Usuários Permitidos
Status da Conta (Ativo / Inativo)

Botão final:

Criar Cliente

Após criar o cliente ele deve aparecer automaticamente na lista de clientes.

3️⃣ Corrigir Botão "Adicionar Usuário Interno"

Ao clicar deve abrir um modal de cadastro de membro da equipe Soccer Mind.

Campos:

Nome
Email
Cargo
Permissão (Admin / PMO / Desenvolvedor / Suporte)
Senha inicial

Botão final:

Criar Usuário
4️⃣ Melhorar Página "Usuários Internos"

Na lista de usuários internos adicionar ações.

Editar usuário

Desativar usuário
5️⃣ Função "Alterar Senha"
